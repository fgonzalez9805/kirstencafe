☁️ Google Drive — sincronización entre dispositivos

Conecta Google Drive para compartir datos entre todos tus dispositivos. Se guarda un archivo kirsten-data.json en tu Drive.

Cómo conectar:
1. Crea un archivo vacío kirsten-data.json en tu Google Drive.
2. Clic derecho → "Obtener enlace" → copia el ID (la parte entre /d/ y /view).
3. Ve a OAuth Playground, autoriza "Drive API v3 → drive.file", genera el token y pégalo.
4. Guarda y sincroniza. Desde cualquier dispositivo repite el paso 3 para obtener un token fresco.
